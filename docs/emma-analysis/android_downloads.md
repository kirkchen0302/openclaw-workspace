## 關鍵觀察

1. **Android 新使用者腰斬**：2025-01 的 39,440 → 2026-02 的 17,253（-56%）
2. **2025-10 起淨流失**：卸載 > 新使用者，每月淨減 2,000~9,000 裝置
3. **活躍裝置半年掉 7 萬**：1,116,035（2025-05）→ 1,036,314（2026-03）
4. **2025-03-25 有異常高峰**：單日 9,451 裝置安裝、7,238 新使用者（可能是廣告投放或活動）

## 各指標建議用哪個來源追蹤

| 用途 | 建議來源 | 原因 |
|---|---|---|
| 新下載/獲客 | **Play Console** | 精確不重複，每天自動匯出到 GCS |
| 日活/月活 | **GA4 API** | 跨平台、即時、含行為事件 |
| MAVS/HYVS | **BigQuery** | 唯一能關聯到會員的 |
| 卸載率 | **Play Console** | 唯一有精確卸載數據的 |
| iOS 下載 | **App Store Connect** | 等 ONE_TIME_SNAPSHOT |
