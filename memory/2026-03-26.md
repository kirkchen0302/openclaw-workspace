## audience_dashboard 定義修正（重要）

### 919 載具（載具驗證失敗）

**定義**：財政部回應代碼 919，代表該會員的手機載具驗證碼錯誤，無法同步發票資料。

**正確判斷方式**：
```sql
-- 新（正確）
JOIN `production-379804.intermediate.sat__carrier` c ON lmc.carrier_hk = c.carrier_hk
WHERE c.mof_status = 'INACCURATE_AVAILABILITY'
```

**舊（錯誤）方式**：
```sql
-- 舊（錯的，不應使用）
JOIN `production-379804.intermediate.hub__carrier` hc ON lmc.carrier_hk = hc.carrier_hk
WHERE hc.barcode LIKE '/%'
```

`hub__carrier` 是識別表，`barcode LIKE '/%'` 是條碼格式判斷，並不代表財政部驗證失敗；應改用 `sat__carrier.mof_status` 判斷。
